#Babel starter

with system.js

to use:

   download file     
1. `git clone git@github.com:cbstodd/babel_starter.git`
    
    install dependencies
2. `npm install` 

    runs babel watcher
3. `run npm build`
 
    starts server
4. `npm start` 

to rename the git repo to your own:
`git remote set-url origin https://github.com/YOUR_USERNAME/YourRepoName.git`
